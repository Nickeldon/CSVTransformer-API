# CSVTransformer-API
Do simple and complex CSV transformations using this API

## Requirements

- NodeJS
- A computer :skull:

## How to use it?

Clone the repository
```
git clone https://github.com/Nickeldon/CSVTransformer-API.git
```

Open the folder
```
cd CSV-Transformer-API
```

Execute the program
```
node main.js
```
